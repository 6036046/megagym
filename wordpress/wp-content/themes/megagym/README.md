# MegaGym Community Theme

Een aangepast WordPress-thema voor de gym community website.

## Kenmerken

- Responsive homepage met hero, voordelen, events en reviews.
- Ondersteuning voor custom logo en menu.
- Shortcode-integratie met de plugin `Gym Community Tools`.
- Ondersteuning voor evenementen- en review-archieven.

## Installatie

1. Upload de map `megagym` naar `wp-content/themes/`.
2. Activeer het thema onder `Weergave > Thema's`.
3. Zorg dat de plugin `Gym Community Tools` actief is.
4. Stel een menu in onder `Weergave > Menu's`.
