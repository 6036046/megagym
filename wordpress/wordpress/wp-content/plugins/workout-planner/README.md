# Mega Workout Planner 💪

Welkom bij de **Mega Workout Planner**! Dit is een custom WordPress plugin, speciaal gebouwd om naadloos aan te sluiten op het zwart met gouden 'Mega Gym' thema. 

Met deze plugin kunnen gebruikers (en jijzelf) eenvoudig een persoonlijk fitness schema samenstellen direct vanaf de website.

## ✨ Features
* **Gebruiksvriendelijk Formulier:** Voeg snel een dag, oefening, sets en reps toe.
* **Ingebouwde Styling:** De plugin bevat zijn eigen inline CSS. Alles is direct in strak Mega Gym zwart (#000) en goud (#D4AF37) opgemaakt. Geen gedoe met losse stylesheets!
* **Live Schema:** Zodra je een oefening toevoegt, verschijnt deze direct in een overzichtelijke tabel onder het formulier.
* **Admin Overzicht:** Bekijk alle ingevulde workouts in je WordPress backend via een speciaal nieuw menu-item.

## 🚀 Installatie
1. Maak een map aan genaamd `mega-workout-planner`.
2. Zorg dat de bestanden `workout-planner.php` en deze `README.md` in de map staan.
3. Selecteer de map (of de bestanden) en comprimeer ze tot een `.zip` bestand.
4. Ga in je WordPress dashboard naar **Plugins > Nieuwe plugin > Plugin uploaden**.
5. Upload het `.zip` bestand en klik op **Nu installeren**.
6. Klik na de installatie op **Activeren**.

## 💻 Hoe te gebruiken
Het is supersimpel! Om de planner en het formulier op je website te tonen, hoef je alleen maar de volgende shortcode op een willekeurige Pagina of in een Bericht te plaatsen:

`[workout_planner]`

Sla je pagina op, en je bent klaar om te trainen!

## 📂 Bestandsstructuur
De plugin is lekker lichtgewicht gehouden en bestaat in de basis uit slechts één bestand:
* `workout-planner.php` (Bevat de database setup, het formulier, de tabel, inline CSS en admin pagina).
